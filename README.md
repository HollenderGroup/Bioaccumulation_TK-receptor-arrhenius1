# Bioaccumulation_TK-receptor-arrhenius1
This project repository is connected to the journal article **‘A Moment of Heat, a Lifetime Exposure? Integrating effects of temperature and heatwaves into Toxicokinetic Modelling of a Neonicotinoid in Aquatic Invertebrates’** by Raths et al. 2026 (doi TBA).

The key element of this repository is the integration of the Arrhenius Equation for temperature effects on k_u and k_e into the previously developed TK-receptor model Raths et al. (2023)* (https://github.com/NikaGoldring/TK-receptor) under dynamic pesticide and temperature exposure (state variables).

The model uses the BYOM platform (https://www.debtox.info/byom.html).

How to run the model simulations?
Navigate to "BYOM/TK_receptor_arrhenius1", run "pathdefine.m", use any script (i.e. "byom_temp_thi_Run1_Amb.m") as your template.

*https://doi.org/10.1021/acs.est.3c01891
